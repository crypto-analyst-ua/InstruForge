Наша діяльність щодо обробки персональних даних регулюється відповідно до Закону України «Про захист персональних даних» та інших чинних нормативно-правових актів України. Ми також дотримуємося Загального регламенту щодо захисту даних (GDPR) для користувачів з Європейського Союзу.

2. Які дані ми збираємо?

Ми збираємо кілька типів інформації для надання та покращення наших послуг:

Дані, надані вами:

  • Ідентифікаційні дані: Ім'я, прізвиче, електронна адреса (email), номер телефону.
  • Дані для доставки: Адреса доставки, місто, відділення поштового оператора (Нова Пошта).
  • Фінансові дані: Дані для оплати (обробляються платіжними провайдерами, а не нами безпосередньо). Ми не зберігаємо дані вашої кредитної картки.
  • Дані облікового запису: Логін, пароль (у зашифрованому вигляді), історія замовлень, перегляди товарів, список бажань.
  • Зв'язок: Листування з нами через email чи інші канали.

Дані, що збираються автоматично:

  • Технічні дані: IP-адреса, тип браузера, версія браузера, часовий пояс, типи та версії доповнень браузера, операційна система та платформа.

  • Cookies (куки): Дивіться розділ «Cookies» нижче.

Ми використовуємо ваші персональні дані для таких цілей:

  • Для обробки та доставки вашого замовлення.

5. Поширення ваших даних

Ми можемо надавати вашу інформацію обмеженому колу третіх осіб для виконання наших зобов'язань:

  • Служби доставки: Такі як «Нова Пошта» для доставки вашого замовлення. Ми надаємо їм лише необхідний мінімум даних (ПІБ, телефон, адресу).
  • Технічні провайдери: Ми використовуємо сторонні сервіси для роботи нашого Сайту:
    • Firebase (Google): Для аутентифікації, зберігання даних (Firestore) та файлів (Storage). Політика конфіденційності Google.
    • Cloudinary: Для оптимізації та зберігання зображень товарів. Політика конфіденційності Cloudinary.
    • EmailJS: Для надсилання листів підтвердження замовлення на нашу пошту. Політика конфіденційності EmailJS.
    Ці провайдери мають доступ до даних, необхідних для виконання своїх функцій, і зобов'язані не використовувати їх для інших цілей.
  • Законні вимоги: Ми можемо розкрити ваші дані, якщо це необхідно для виконання законного запиту правоохоронних органів або для захисту наших прав.

Наш Сайт використовує cookies — невеликі текстові файли, які зберігаються на вашому пристрої. Вони допомагають:

  • Запам'ятовувати товари в вашому кошику та ваші вподобання.
  • Забезпечувати безпековий вхід в обліковий запис.
  • Аналізувати трафік та роботу Сайту для покращення послуг.

Ви можете налаштувати свій браузер так, щоб він відмовляв від усіх або деяких файлів cookie. Однак це може вплинути на функціональність Сайту.
